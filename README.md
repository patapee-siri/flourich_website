# 🌸 Flourich: พื้นที่ปลอดภัย เพื่อใจที่เข้มแข็ง

**Flourich** (ฟลูริช) คือแพลตฟอร์มดูแลสุขภาพจิตที่ออกแบบมาเพื่อเป็น "พื้นที่ปลอดภัย" สำหรับคนรุ่นใหม่ (วัยเรียนและวัยทำงาน) ที่เผชิญกับความเครียด ความกดดัน หรือต้องการที่ระบายโดยปราศจากการตัดสิน

---

## 🚀 แนวคิดของโปรเจกต์ (Concept)
จากการสำรวจกลุ่มตัวอย่าง 105 คน พบว่าหลายคนรู้สึก "โดนมองข้าม" หรือ "ไม่กล้าเล่าเพราะกลัวโดนตัดสิน" เราจึงสร้าง Flourich ขึ้นมาเพื่อเป็นทางออก:
* **ความโดดเดี่ยว:** เปลี่ยนความรู้สึกที่ต้องเผชิญปัญหาเพียงลำพัง ให้มีพื้นที่รับฟังที่เข้าใจจริงๆ
* **การเข้าถึง:** ลดช่องว่างในการเข้าถึงบริการดูแลใจให้ง่ายและเป็นมิตรมากขึ้น
* **ความปลอดภัย:** ให้ความสำคัญกับความเป็นส่วนตัวสูงสุด (Privacy-First)

## ✨ ฟีเจอร์หลัก (Key Features)

### 🛡️ Safe Room
พื้นที่ระบายความรู้สึกส่วนตัวผ่านข้อความ เสียง หรือวิดีโอ มีระบบ **Mirror Mode** ที่ใช้ AI ช่วยสะท้อนอารมณ์และให้คำแนะนำเบื้องต้นเพื่อประคองใจ

### 🗣️ Echo Room
Community ของกลุ่มคนที่มีประสบการณ์ใกล้เคียงกัน (Empathy Match) เพื่อแบ่งปันเรื่องราวและให้กำลังใจซึ่งกันและกัน โดยเน้นการไม่ระบุตัวตน

### 🌱 Inner Child (Avatar Hamster)
ตัวแทนเด็กน้อยภายในใจที่จะเติบโตไปพร้อมกับคุณ เมื่อคุณทำภารกิจพัฒนาใจหรือบันทึกอารมณ์สม่ำเสมอ ตัวละครหนูแฮมสเตอร์ของคุณจะค่อยๆ เติบโตขึ้น

## 🛠️ Tech Stack
โปรเจกต์นี้พัฒนาด้วยเทคโนโลยีสมัยใหม่:

* **Styling:** Tailwind CSS
